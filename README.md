# CyberSecurity News Curation

## 概要

このリポジトリでは、サイバーセキュリティに関する最新のニュースやトピックをキュレーションし、整理して掲載しています。

## 作成にあたって

- 収集にあたって，なるべく1次ソースを追うようにしています。
- 主に収集するニュース
  + 日本との関係が強いと考えられるニュース
  + 攻撃者の動向（攻撃手法，狙いなど）
- あまり収集しないニュース
  + 製品の脆弱性情報

## コンテンツ構成

- ページタイトル
- サマリー（日本語）
- タグ
- 収集日
- ページURL

## 免責事項

当リポジトリは情報提供を目的としており、記載された内容の正確性・完全性を保証するものではありません。
本リポジトリに掲載した内容は予告なく変更することがあります。

# 2026年2月 サイバー脅威動向レポートサマリー
## 1. AI技術に関連する攻撃と悪用
AIシステムの普及に伴い、モデル自体の侵害やAI機能を攻撃工程に組み込む動きが確認されています。

* **AIモデルの蒸留攻撃**: 中国のAI企業が、数千万件のやり取りを通じてAnthropic社のモデルから能力を不正に抽出し、自社モデルの改善に利用しようとしたと報じられています。
* **AIエージェントの侵害**: 人気のあるAIエージェントである `OpenClaw` の拡張機能を装い、ユーザーからパスワードを盗み出すマルウェアが確認されているという記事が出ています。
* **AIによる攻撃プロセスの高速化**: ロシア語圏のアクターが生成AIサービスを利用し、わずか5週間で55カ国の `FortiGate` デバイス600台以上を侵害したと伝えられています。
* **推薦機能の操作**: ウェブサイト上の「AIで要約」ボタンに隠された命令を埋め込み、AIアシスタントの記憶や推奨内容を攻撃者に有利なように操作する手法が特定されたと報じられています。

## 2. 開発者およびサプライチェーンへの攻撃
ソフトウェア開発の工程やエコシステムを標的にした活動が続いています。

* **偽の採用活動**: 北朝鮮のアクターなどが求人担当者を装い、開発者に技術テストとして悪意あるコードを実行させる手法が横行しているとのことです。
* **リポジトリの汚染**: `npm` や `PyPI` などの公開リポジトリに、暗号資産を盗むコードやバックドアを含むパッケージが多数投稿されていると報じられています。
* **更新メカニズムの悪用**: テキストエディタ `Notepad++` の更新機能を乗っ取り、特定の対象にマルウェアを配信する攻撃が行われたという記事が出ています。

## 3. 検出回避技術とインフラの多様化
攻撃者は、インフラの遮断を免れるために多段階の手法や新しい通信チャネルを利用しています。


* **ブロックチェーンの悪用**: ボットネット `Aeternum` が、 `Polygon` ブロックチェーンのスマートコントラクトを指令サーバーとして利用し、遮断を困難にしていると伝えられています。
* **正規クラウドサービスの転用**: `Zoho WorkDrive` や `Google Drive` などの正規サービスを、マルウェアの指令送受信やデータ窃取の場として悪用するケースが増えているとのことです。

* **DNSを介した指令受領**: `nslookup` コマンドを通じて `DNS` クエリからマルウェアのペイロードを取得する攻撃手法が確認されたと報じられています。

## 4. 国家の関与が疑われるサイバー活動
特定の国に関連するグループが、重要インフラや隔離環境を標的に活動しています。


* **物理的隔離ネットワークの突破**: 北朝鮮の `APT37` が、 `USB` メモリなどのリムーバブルメディアを介して、隔離されたネットワークに侵入しデータを窃取するツールを使用しているという記事が出ています。
* **ネットワーク機器の監視**: 中国に関連するアクターが、ルーターなどのエッジデバイスで動作するフレームワークを使い、通信の監視や改ざんを行っていると伝えられています。
* **組織的な政府・インフラ攻撃**: シンガポールの通信セクターやヨーロッパの政府機関を狙った、継続的な標的型攻撃が発生しているとのことです。

## 5. データ漏洩とランサムウェアの変容
大規模な情報の流出が続く一方で、被害者の対応や標的に変化が見られます。

* **数千万件規模の個人情報流出**: DIYプラットフォームの `ManoMano` から3,800万人分、米国の自動車サイト `CarGurus` から1,240万人分のデータが流出したと報じられています。
* **身代金支払率の低下**: ランサムウェア攻撃件数が増加している一方で、身代金を支払う被害者の割合は28パーセントまで低下したという記事が出ています。
* **金融機関への攻撃集中**: 金融組織の65パーセントがランサムウェアの標的となっており、これは他業界と比較して高い割合であると指摘されています。
